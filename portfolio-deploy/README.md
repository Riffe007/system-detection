# System Monitor - Download

## About This Application

A real-time system monitoring application built with **Tauri**, **React**, and **Rust**. This desktop application provides comprehensive monitoring of your computer's performance, security, and optimization metrics.

## Features

- **Real-time System Monitoring**: CPU, Memory, Network, and Disk usage
- **Security Monitoring**: Threat detection and security scoring
- **Performance Optimization**: Bottleneck identification and recommendations
- **Cross-platform**: Built with Tauri for native performance
- **Modern UI**: Clean, responsive interface with real-time updates

## Download Options

### 1. Standalone Executable (Recommended)
- **File**: `system-monitor.exe` (3.4MB)
- **Type**: Single executable file
- **Best for**: Quick demos, no installation required
- **Usage**: Download and run directly

### 2. Windows Installer
- **File**: `System-Monitor-Setup.exe` (1.3MB)
- **Type**: Professional installer
- **Best for**: Permanent installation with start menu shortcuts
- **Usage**: Run installer and follow setup wizard

## System Requirements

- **OS**: Windows 10/11 (64-bit)
- **RAM**: 4GB minimum
- **Storage**: 100MB free space
- **Permissions**: Administrator access recommended for full monitoring capabilities

## Installation Instructions

### For Standalone Executable:
1. Download `system-monitor.exe`
2. Right-click the file and select "Run as administrator" (if prompted)
3. If Windows shows a security warning, click "More info" then "Run anyway"
4. The application will start and begin monitoring your system

### For Windows Installer:
1. Download `System-Monitor-Setup.exe`
2. Run the installer as administrator
3. Follow the installation wizard
4. Launch from Start Menu or desktop shortcut

## Security Note

This is an unsigned executable built for demonstration purposes. Windows may show security warnings - this is normal for development builds. The application only monitors system metrics and does not collect or transmit any personal data.
